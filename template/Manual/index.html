A) HTML Structure (web version) - top

This template have a liquid layout width at 100%. All the elements in the code are named with appropriate IDs and Classes, so you can easily find what you need to edit in the HTML and the CSS.

How to change header background color?
You can change header background color by changing color code of #header_wrapper in /css/screen.css. For example: 
body #header_wrapper
{
	background: #000000 url('../images/header_bg.png') no-repeat top center;
	padding: 20px 0 0 0;
	height: 236px;
}
body.home #header_wrapper
{
	background: #000000 url('../images/header_bg.png') no-repeat top center;
	padding: 20px 0 0 0;
	height: 550px;
}

How to change background pattern?
You can change header background pattern by changing class name of #header_pattern in /css/screen.css. Open /css/screen.css find "Predefined background pattern" to see all possible pattern options.

B) CSS Files and Structure - top

I'm using five CSS files in this theme. The first file contains all of the specific stylings for the page. The file is separated into sections using:

	
	/* CSS Reset */

	some code

	/* Global layout */
	
	some code
	
	/* Input layout */
	
	some code


The second file is a styling for jQuery UI element.

The third file is a styling for modal window.

The fourth and fifth file is a styling for HTML5 video.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

C) JavaScript - top

This theme imports ten Javascript files.

  1. jQuery
  2. jQuery UI
  3. jQuery Validate
  4. jQuery fancybox
  5. jQuery anythingSlider
  6. jQuery nivoslider
  7. jQuery coinslider
  8. jQuery easing
  9. Browser check
  10. Custom script
  1. jQuery and jQuery UI are a Javascript library that greatly reduces the amount of code that you must write.
  2. jQuery validate contact form
  3. jQuery Fancybox create a nice popup window using in gallery
  4. jQuery anythingSlider for homepage and portfolio slider
  5. jQuery nivoslider for homepage slider
  6. jQuery coinslider for homepage slider
  7. jQuery easing for slider transition effect
  8. Browser check check browser name and version
  9. Most of the animation in this theme is carried out from the customs scripts.
